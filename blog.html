FAQ!

a. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web document as a tree of objects. Each node in the tree corresponds to a part of the document (such as elements, attributes, and text), allowing programming languages like JavaScript to interact with and manipulate the content and structure of a webpage dynamically.

b. How do you select an element from the DOM?

I can select an element from the DOM using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. These methods allow you to access and manipulate specific elements based on their ID, class, tag, or CSS selectors.

d. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use `element.setAttribute('name', 'value')` to set and `element.getAttribute('name')` to get an attribute. For styles, use `element.style.propertyName` to set a style, like `element.style.backgroundColor = 'red'`. This allows dynamic changes to elements in your web application.
